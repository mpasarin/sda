SDA - Software development assistant

Run commands
  sda <command> ... <param?> ...

  Run one command or many commands at once. Parameters affect to all commands.

  sda <env> <command>             - Runs the commands in a specific environment.
  sda [-a | --all] <command>      - Runs the commands in all environments.

Setup environments
  sda setup <template> <folder?>  - Set up a new environment in the specified folder, or the current folder.
  sda attach <template> <folder?> - Attach a template to a folder, or the current folder.

Explore your configuration
  sda list                        - Lists all environments in SDA.
  sda <env> list                  - Lists all commands for an environment.

Get help
  sda help
